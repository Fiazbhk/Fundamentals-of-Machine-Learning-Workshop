# Fundamentals of Machine Learning — Workshop

> A hands-on beginner workshop covering the **ML Lifecycle**, **core algorithms**, and real-world data exploration using the Titanic and Diabetes datasets.

## What You'll Learn

- What **Machine Learning** is and how it differs from traditional programming
- The difference between **AI, ML, and Deep Learning**
- The three types of ML: **Supervised, Unsupervised, and Reinforcement Learning**
- The complete **ML project lifecycle** from data collection to deployment
- How to **clean, preprocess, and explore real-world datasets**
- How to implement core **ML algorithms** using Python and scikit-learn
- How to **evaluate model performance** using accuracy, precision, and recall

## Datasets

We use two classic datasets for hands-on practice:

[![Kaggle Dataset](https://img.shields.io/badge/Kaggle-Titanic%20Dataset-20BEFF?style=for-the-badge&logo=kaggle)](https://www.kaggle.com/datasets/yasserh/titanic-dataset)

> The Titanic dataset contains passenger information such as age, sex, ticket class, and survival status — used for data cleaning and preprocessing.

[![Kaggle Dataset](https://img.shields.io/badge/Kaggle-Diabetes%20Dataset-20BEFF?style=for-the-badge&logo=kaggle)](https://www.kaggle.com/datasets/mathchi/diabetes-data-set)

> The Diabetes dataset is used for correlation analysis, outlier detection, and normalization.

## Getting Started

### Option 1 — Open in Google Colab (Recommended, no setup needed)

[![Open in Colab](https://colab.research.google.com/assets/colab-badge.svg)](https://colab.research.google.com/drive/1TrK2kAzclMm-F_NuIC2fIW5ESR4_P_fL?usp=sharing)

### Option 2 — Run Locally

**1. Clone the Repository**

```bash
git clone https://github.com/Fiazbhk/Fundamentals-of-Machine-Learning-Workshop.git
cd Fundamentals-of-Machine-Learning-Workshop
```

**2. Install Dependencies**

```bash
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

**3. Launch the Notebook**

```bash
jupyter notebook Workshop_Fundamentals_of_Machine_Learning.ipynb
```
